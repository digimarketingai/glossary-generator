# CSV Glossary Generator | CSV 詞彙表產生器

A simple tool that converts CSV files into beautiful, searchable glossary websites.

一個簡單的工具，可將 CSV 檔案轉換為美觀、可搜尋的詞彙表網站。

---

## Features | 功能特色

- **CSV Upload** | **CSV 上傳** — Drag and drop or click to upload your glossary CSV file. 拖放或點擊上傳您的詞彙表 CSV 檔案。

- **Live Preview** | **即時預覽** — See your glossary website before downloading. 下載前即時預覽您的詞彙表網站。

- **Search Function** | **搜尋功能** — Generated websites include a search bar to filter terms. 產生的網站包含搜尋列，可篩選詞彙。

- **Responsive Design** | **響應式設計** — Works on desktop, tablet, and mobile devices. 適用於桌機、平板和手機。

- **One-Click Download** | **一鍵下載** — Download your glossary as a standalone HTML file. 將詞彙表下載為獨立的 HTML 檔案。

---

## How to Use | 使用方法

### Step 1 | 步驟一
Prepare a CSV file with your glossary terms. The first row should be headers.

準備一個包含詞彙的 CSV 檔案。第一行應為標題列。

**Example CSV | CSV 範例：**
```
Term,Definition,Category
API,Application Programming Interface,Technology
SEO,Search Engine Optimization,Marketing
UI,User Interface,Design
```

### Step 2 | 步驟二
Open the tool and upload your CSV file by dragging it into the upload area or clicking to browse.

開啟工具，將 CSV 檔案拖曳至上傳區域，或點擊瀏覽選擇檔案。

### Step 3 | 步驟三
Preview your glossary website in the preview panel.

在預覽面板中檢視您的詞彙表網站。

### Step 4 | 步驟四
Click "Download HTML" to save your glossary website.

點擊「Download HTML」下載您的詞彙表網站。

## CSV Format Requirements | CSV 格式要求

| Requirement | 要求 | Description | 說明 |
|-------------|------|-------------|------|
| Headers | 標題列 | First row must contain column headers | 第一行必須包含欄位標題 |
| Encoding | 編碼 | UTF-8 recommended for multilingual content | 建議使用 UTF-8 以支援多國語言 |
| Delimiter | 分隔符號 | Comma-separated values | 以逗號分隔的值 |
